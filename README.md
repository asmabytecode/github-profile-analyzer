# GitHub Profile Analyzer

A small web tool that shows GitHub user information in a simple and readable way.

You enter a GitHub username, and it pulls data from the GitHub API and displays it on the page.

## What it shows

- basic profile info (avatar, name, bio)
- number of repositories
- number of followers
- total stars across repositories
- most used programming languages
- a simple chart for language distribution

---

Built as a small practice project to work with APIs and data visualization.
